libprs500 User Manual

libprs500 is an e-book library manager. It can view, convert and catalog e-books in most of the major e-book formats. It can also talk to a few e-book reader devices. It can go out to the internet and fetch metadata for your books. It can download newspapers and convert them into e-books for convenient reading. It is cross platform, running on Linux, Windows and OS X.

To get started you should use the Graphical User Interface. It can be launched using its icon or the command libprs500.

For advanced usage you should check out the Command Line Interface.

You may find the list of Frequently Asked Questions useful as well.
